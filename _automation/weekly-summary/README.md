# Weekly Summary Skill

This skill generates a concise weekly progress summary for the `pharma_skills` repository and posts it to Slack.

## Usage

Trigger this skill to:
1. Summarize recent commits, PRs, and issues.
2. Post a developer-focused update to Slack.
3. Save a markdown record of the summary.

- `SKILL.md`: The core skill definition and workflow.
